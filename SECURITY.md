# Security Policy

## Reporting a vulnerability

If you find a security vulnerability in this repo, please report it to us via GitHub Security Advisories below:

https://github.com/splameiplay/splameiplay-studio/security/advisories/new

Please **do not** open and issue or pull request for security issues so we can keep everyone safe!

## How we'll respond

- We'll acknowledge your issue within 72 hours of it being created
- We'll look into the issue and determine the impact
- If we accept it, we'll on fixing the issue and disclosing it

# How we'll disclose issues

To help protect everyone and prevent any major issues, we ask if you can keep vulnerabilities kept private between us until it has been fixed **and** in a full release or we've got a disclosure date.

## Credits

We're very happy to credit those who report security vulnerabilities unless you wish to stay anonymous.
